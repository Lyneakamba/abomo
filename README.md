# INSIDE COMMERCIAL SARL — Site Vitrine

Site vitrine de **INSIDE COMMERCIAL SARL**, société camerounaise spécialisée dans la distribution de produits à base de gypse/plâtre, membre du groupe **THE INSIDE GROUP**.

Partenaire stratégique : **AL ALAMIA** (Égypte), leader de la production de plâtre depuis 2005.

## Stack technique

- **Framework** : Laravel
- **Templating** : Blade
- **CSS** : Tailwind CSS v3
- **Assets** : Vite

## Installation

```bash
composer install
npm install
cp .env.example .env
php artisan key:generate
npm run build
php artisan serve
```
